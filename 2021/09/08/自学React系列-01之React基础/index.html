JSX语法

  • 定义虚拟 DOM 的时候不要写引号

  • 标签中混入 JS 表达式时要用 {}

  • 使用样式时，类名不要用 class 而要用 className

  • 内联样式，使用 style 属性时要用 {{style: value}}，如果有样式是 background-color 这种的改写成坨峰形式即 backgroundColor

  • 只有一个顶级标签(要标签)

  • 标签必须闭合

  • 标签首字母小写会转为同名的 html 标签

  • 标签首字母大写代表是 React 的组件
